# Qtrip Project
## How to run?
### 1. Clone this repo
### 2. Open qtrip-frontend folder in terminal
### 3. Run npm install
### 4. Run npm run dev
### 5. Open qtrip-backend folder in terminal
#### [add your .env file with the variables imported in config folder]
### 6. Run npm install
### 7. Run node server.js
